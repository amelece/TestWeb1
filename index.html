Kas ir div tags?

div sadala HTML dokumentu.
To izmanto kā konteineru HTML elemntiem, ko pēc tam stilizē ar CSS vai JS.
Tagam var pievinot klasi vai id atribūtus, tādejādi stilizējot tā saturu.
div tagā var likt jebkādu saturu.
Pirms un pēc div taga automātiski tiek atstāta viena brīva rindkopa.
Tas ir plaši pieņemts, tā ka internetpārlūkprogrammas, kā, piemēram, Google Chrome, Microsoft Edge, Mozilla Firefox, Safari, Opera, to atpazīst.

Kas ir span tags?

span izmanto, lai izceltu vai iekrāsotu daļu teksta vai dokumenta.
Tagu stilizē ar CSS vai JS, pievienojot klasi vai id atribūtu.
span ir līdzīgs div tagam, tomēr tas neatdala tā saturu jaunā rindkopā.
Tas ir plaši pieņemts, tā ka internetpārlūkprogrammas, kā, piemēram, Google Chrome, Microsoft Edge, Mozilla Firefox, Safari, Opera, to atpazīst.

Kā pielietot tagus?

Tags ir īpaša vārdu vai burtu kombinācija, kas pierakstīta starp < un >
Ar tagiem var stilizēt savu dokumentu, atdalot rindkopas vai pievienojot linkus, vai izveidojot tabulas, lai gan tās netiek vairs tik bieži lietotas.
